# Plan 3 — 사이트 확장: 네비게이션 + 프로젝트 페이지

## 개요

개인 블로그를 포트폴리오 겸 멀티 서비스 허브로 확장한다.

---

## 추가 예정 페이지

| 경로 | 내용 |
|------|------|
| `/apps` | 앱 목록 (웹앱 + 앱스토어 앱) |
| `/apps/timelens` | 공부 타이머 + 통계 웹앱 소개 페이지 |
| `/apps/tennis-counter` | Tennis Counter 앱스토어 앱 소개 페이지 |
| `/playground` | 프론트엔드 기술 위키 + 인터랙티브 데모 |

---

## 프로젝트 상세

### TimeLens (웹앱)
- 공부 시간 타이머
- 실제 공부한 시간 통계 시각화
- 별도 웹앱으로 개발 예정, 블로그에서 소개 페이지만 제공

### Tennis Counter (앱스토어 앱)
- 테니스 경기 중 점수 카운터 앱
- 앱스토어 링크, 기능 소개, 스크린샷

### Playground
- 프론트엔드 기술 위키
- 카테고리별 인터랙티브 데모 (애니메이션, 폼, 레이아웃 등)
- 각 항목: 라이브 데모 + 코드 스니펫 병렬 노출

---

## 네비게이션 구조

```
[블로그]  [Apps]  [플레이그라운드]      (우측: GitHub 아이콘 등)
```

- 데스크탑: 상단 네비게이션 바 (전체 메뉴 노출)
- 모바일: 햄버거 버튼 → 드로어 또는 드롭다운

---

## 구현 순서 (예정)

1. 공통 네비게이션 바 컴포넌트 추가 (`src/components/nav/`)
2. `/apps` 목록 페이지
3. `/apps/[slug]` 동적 라우트 (각 앱 상세)
4. `/playground` 기술 위키 페이지
