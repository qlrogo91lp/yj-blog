# Roadmap

블로그 기능 개선 로드맵. 누락된 기능과 보완 사항을 정리한다.

---

## 관리자 페이지 (`/admin`)

### 1. 글 관리 (`/admin/posts`)

| 항목 | 상태 | 위치/설명 |
|------|------|-----------|
| 글 수정 진입점 | ⚠️ 누락 | 목록 테이블에 편집 버튼 없음. `/admin/posts/[id]/edit` 페이지는 구현 완료 → 테이블 행에 편집 링크 추가 |
| 글 삭제 기능 | ❌ 미구현 | Server Action + UI 버튼 + 확인 다이얼로그 필요 |
| 글 상태 일괄 변경 | ❌ 미구현 | 체크박스 선택 → 일괄 발행/비공개 전환 |
| 글 목록 검색/필터 | ❌ 미구현 | 제목·카테고리 기반 검색/필터링 |

### 2. 대시보드 (`/admin`)

| 항목 | 상태 | 설명 |
|------|------|------|
| 최근 글 목록 | ❌ 미구현 | 최근 작성/수정된 글 5건 표시 |
| 최근 댓글 목록 | ❌ 미구현 | 새 댓글 빠른 확인용 |
| 오늘 조회수 요약 | ❌ 미구현 | 통계 페이지에는 있지만 대시보드에도 요약 표시 |
| 빠른 작업 버튼 | ❌ 미구현 | "새 글 작성" 바로가기 등 |

### 3. 통계 (`/admin/statistics`)

| 항목 | 상태 | 설명 |
|------|------|------|
| 유입 경로 페이지 | ⚠️ placeholder | `/admin/statistics/referrers` — 페이지만 존재, 데이터 표시 없음 |
| 인기 글 순위 | ❌ 미구현 | 조회수 기반 인기 글 Top 10 |
| 글별 조회수 상세 | ❌ 미구현 | 각 글의 일별 조회수 추이 |

### 4. 설정 (`/admin/settings`)

| 항목 | 상태 | 설명 |
|------|------|------|
| 블로그 설정 | ⚠️ placeholder | "준비 중입니다." 텍스트만 표시 |
| 고려 항목 | - | 블로그 이름, 소개글, 소셜 링크, SEO 기본값 등 |

## 공개 페이지 (블로그 메뉴)

현재 네비게이션: `Blog` | `Apps` | `Playground`

| 항목 | 상태 | 설명 |
|------|------|------|
| 검색 기능 | ❌ 미구현 | 글 제목/내용 검색. Header 또는 `/posts` 페이지에 추가 |
| Playground | ⚠️ placeholder | "곧 업데이트" 메시지만 표시 |
| 태그 시스템 | ❌ 미구현 | 현재 카테고리(1:N)만 지원. 다중 태그 필요 시 posts_tags 중간 테이블 추가 |
| 글 목차(TOC) | ❌ 미구현 | 글 상세 페이지 heading 기반 사이드 목차 |
| 시리즈/연재 | ❌ 미구현 | 관련 글 묶기 기능 |
| RSS 피드 | ❌ 미구현 | `/feed.xml` 또는 `/rss` 엔드포인트 |
| 이전/다음 글 네비게이션 | ❌ 미구현 | 글 상세 하단에 이전/다음 글 링크 |
| 글 공유 버튼 | ❌ 미구현 | 링크 복사, SNS 공유 등 |
| 다크모드 | ⚠️ 부분 구현 | CSS 변수(`.dark`)·shadcn `dark:` 스타일은 존재하나, 테마 전환 라이브러리(`next-themes`) 및 토글 UI 미구현 |

---

## 우선순위 제안

1. 다크모드 (`next-themes` + 토글 UI)
2. 글 목록에 수정/삭제 버튼 추가 (`/admin/posts` 테이블 Actions 컬럼)
3. 글 삭제 Server Action (`delete-post.ts`)
4. 대시보드 최근 활동 (최근 글 + 최근 댓글 위젯)
5. 검색 기능 (공개 페이지 글 검색)
6. 글 상세 TOC (긴 글 탐색 UX 개선)
7. RSS 피드 (블로그 기본 기능)
